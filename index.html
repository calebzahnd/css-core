Core

Core is a very simple, lightweight alternative to larger frameworks such as Bootstrap and grid systems like Susy. It is an opinionated system, built with the belief that markup should be mean and lean and layout should be controlled in stylesheets, not a bunch of classes.

Easy to manage media queries

In Core, we try to limit breakpoints to a just a few predefined widths. These are easily managed via a Sass Map in the base/_variables.scss partial.

    $grid-breakpoints: (
        xs: 0,
        sm: 544px,
        md: 768px,
        lg: 992px,
        xl: 1200px
    );

Media queries are dead simple to call in your Sass using the screen() mixin. Simply add @include screen(md){} to your declarations and add your declarations inside the brackets.

Bird's Eye Z-Index

Z-Index is one of the easiest way to get a mess started in your code. With Core, we manage Z-Index easily by calling them all from a Sass Map. All z-indexes should be named in the $z-index map:

    $z-index: (
        heaven      :   9999,
        modal       :   5000,
        aboveground :   100,
        ground      :   1,
        zero        :   0,
        underground :   -1,
        hell        :   -9000
    );

Z-indexes are then easily called using @include z-index(ground);
